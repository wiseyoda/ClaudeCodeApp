# Design Decisions

| Area | Decision |
|------|----------|
| Navigation | NavigationSplitView (sidebar + detail) |
| Settings | Grouped Form in sheet (iOS Settings-style) |
| iPad | Full multitasking support (Stage Manager, Split View, Slide Over) |
| Modals | Sheets for quick actions, push for main flows |
| State | @Observable + actors (Swift 6) |
| Design | Liquid Glass throughout |
| Messages | 3 card types: Chat, Tool, System |
| Interactions | Unified StreamInteractionHandler |
| Platform | Widgets, Control Center, Live Activities, Siri |
