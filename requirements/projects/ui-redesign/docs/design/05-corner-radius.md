# Corner Radius


| Token | Value | Use |
|-------|-------|-----|
| `xs` | 4pt | Tiny elements |
| `sm` | 6pt | Badges, chips |
| `md` | 10pt | Standard cards |
| `lg` | 14pt | Large cards |
| `xl` | 18pt | Sheets |
| `xxl` | 24pt | Modals |
| `pill` | 100pt | Pill buttons |

**Note**: iOS 26 `.glassEffect()` has built-in corner radii. Use glass effects for consistent corners.
